Примеры для семинаров по компьютерной графике в МФТИ
====================================================

Для сборки проекта необходимы зависимости:
* GLFW (http://www.glfw.org/)
* GLEW (http://sourceforge.net/projects/glew/)
* GLM (http://glm.g-truc.net/0.9.5/index.html)
* GL Image (http://glsdk.sourceforge.net/docs/html/index.html)
* Assimp (http://assimp.sourceforge.net/)
* AntTweakBar (http://anttweakbar.sourceforge.net/doc/)

Сборка осуществляется с помощью CMake (http://cmake.org/)

Под Windows будет сгенерировано решение Visual Studio. Нужно открыть решение, нажать правой кнопкой мыши на проекте INSTALL и выбрать пункт Собрать (Build).

Под Linux будут сгенерированы make-файлы. Нужно выполнить команду make install.

Команда install кроме сборки исполняемого файла копирует шейдеры в выходную папку для корректной работы программы. Остается только запустить исполняемый файл.

Список семинаров:
1) Загрузка 3д-моделей на видеокарту с индексами и без индексов.
2) Преобразование вершин. Матрицы модели, вида, проекции.
3) Язык GLSL
4) Освещение
5) Текстурирование
6) Кубические текстуры. Буфер цвета (прозрачность). Буфер глубины (z-fighting). Буфер трафарета
7) Проективные текстуры. Рендеринг в текстуру
8) Тени. Отложенного освещение
9) Постобработка: Gamma correction, Tone mapping, Bloom, SSAO, Depth of Field
